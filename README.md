# Snake
Simple snake game from the project based learning repository

I made this project to develop my skills on DOM manipulation and JavaScript. The idea to buil the project came from the repository "project based learning" which is a list of programming tutorials in which aspiring software developers can learn how to build an application from scratch. This project was also my first time using the canvas element in JS and I used a tutorial to help me with the unfamiliar syntax.

I initially built this project on Codepen and then copied it over to Github.
Codepen Link: https://codepen.io/OtherworldlyGod/pen/dPPzOmG

Link to project based learning repo: https://github.com/practical-tutorials/project-based-learning
